## Goal
Build the simplest possible ERP using Google Sheets and Google Apps Script and deliver a working MVP as early as possible.

## Current Decisions
- Maximum 250 customers
- Maximum 250 tiffins/day
- Primary users: Jay and parents
- Subscription module will be added
- Session is independent of Tiffin Plan
- DailyLog is editable
- GitHub is the source of truth for project documentation
